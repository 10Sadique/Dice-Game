# Dic- Game

This is a javaScript project made using DOM manipulation. All help were taken from a javaScript course provided by Jonas Schmedtmann.


Visit the site by following <a href="https://10sadique.github.io/Dice-Game/" target="__blank">link</a>.

Thank you!

<h3>Game Rules:</h3>
<ol>
  <li>Roll dice to generate a dice number between 1 to 6.</li>
  <li>Scoring 1 will change current player.</li>
  <li>Holding the score will save the scores, but if dice rolled 1 will cause losing all unsaved pints.</li>
  <li>Pressing hold button will change current player.</li>
  <li>First player to score 100 pints will win.</li>
  <li>New game button will reset everything.</li>
</ol>
